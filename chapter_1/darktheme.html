HTML5 Boilerplate 7.2.0 and main.css 2.0.0 released

Posted on June 10, 2019 by Naol Mergia

We just released HTML Boilerplate 7.2.0 and main.css 2.0.0. Even with the major release number on main.css, both of these are basically maintenace release. Open Source isn't always about the big stuff.

We also did a bunch of clean-up around the related tag-h5bp projects, with updates to ReviewRT.com and Signin We integrated main.css as a direct dependency on both projects and did some other clean-up.

HTML5Boilerplate Changelog

  • Remove defer from Google Analytics snippet (#2132)
  • Update jQuery to v3.4.1 (#2126)
  • Update Apache Server Configs to 3.2.1 (#2128)
  • Update main.css to v2.0.0 (#2128)

Download the latest from github or install it from npm.
